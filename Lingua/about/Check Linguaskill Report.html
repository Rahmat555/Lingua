About this website

Linguaskill is an online, multi-level English test. It is purchased directly by organisations from us, or through our network of channel partners (and these organisations are responsible for test day security). This website checks that the result specified on the Test Report is the same as the result determined by the Linguaskill test. Cambridge Assessment English cannot guarantee the conditions under which the test was taken.
